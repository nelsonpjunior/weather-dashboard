## Weather service dashboard

![preview](https://cloud.githubusercontent.com/assets/771149/22400044/12561108-e5ab-11e6-9c3d-8bd3260bf57b.png)

### Requirements

- NodeJS/Npm
- bower
- Gulp

### Instalation

```bash
$ npm install
$ bower install
```

### Running

```bash
$ gulp serve
```

### Running production version

```bash
$ gulp serve:dist
```

### Complile production version

```bash
$ gulp build
```
